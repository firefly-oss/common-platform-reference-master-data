feat: Add Document Template Catalog with localization support

This commit adds a comprehensive Document Template Catalog system similar to the existing Notification Message Catalog. The implementation includes:

1. Database Changes:
   - Added document_template_type_catalog table for storing template types (PDF, DOCX, HTML, etc.)
   - Added document_template_catalog table for storing document templates with template variables
   - Added document_template_localization table for storing localized versions of templates
   - Added template_variables column to notification_message_template table

2. Entity Classes:
   - Added DocumentTemplateTypeCatalog entity
   - Added DocumentTemplateCatalog entity with template variables support
   - Added DocumentTemplateLocalization entity
   - Updated NotificationMessageTemplate entity to support template variables

3. Repository Interfaces:
   - Added DocumentTemplateTypeCatalogRepository
   - Added DocumentTemplateCatalogRepository
   - Added DocumentTemplateLocalizationRepository

4. DTO Classes:
   - Added DocumentTemplateTypeCatalogDTO
   - Added DocumentTemplateCatalogDTO with template variables support
   - Added DocumentTemplateLocalizationDTO
   - Updated NotificationMessageTemplateDTO to support template variables

5. Service Interfaces and Implementations:
   - Added DocumentTemplateTypeCatalogService and implementation
   - Added DocumentTemplateCatalogService and implementation
   - Added DocumentTemplateLocalizationService and implementation
   - Updated NotificationMessageTemplateService to support template variables

6. Mapper Classes:
   - Added DocumentTemplateTypeCatalogMapper
   - Added DocumentTemplateCatalogMapper with JSON conversion for template variables
   - Added DocumentTemplateLocalizationMapper
   - Updated NotificationMessageTemplateMapper to handle template variables

7. Controllers:
   - Added DocumentTemplateTypeCatalogController
   - Added DocumentTemplateCatalogController
   - Added DocumentTemplateLocalizationController

8. Documentation:
   - Added detailed README.md for template systems with usage examples
   - Updated main README.md to include Document Template Catalog API sections

This implementation provides a flexible way to manage document templates with:
- Support for different template types (PDF, DOCX, HTML, etc.)
- Template variables for dynamic content generation
- Localization support for multiple languages
- Version tracking
- Full CRUD operations via REST API

Both Notification Templates and Document Templates now have similar capabilities, making the system more consistent and easier to use.
